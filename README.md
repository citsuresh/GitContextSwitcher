Git Context Switcher

This repository contains the initial scaffold for the Git Context Switcher desktop application.

Projects:
- src/GitContextSwitcher.Core: Core models and service interfaces
- src/GitContextSwitcher.Infrastructure: Infrastructure implementations (stubs)
- src/GitContextSwitcher.UI: WPF UI

Follow-up: open the solution GitContextSwitcher.sln in Visual Studio 2022/2023 or run dotnet build from the src folder.
